# JokiBoy

Landing page jasa tugas kuliah — **1 Hari Pasti Beres**.

## Fitur

- 13 layanan tugas kuliah (Makalah, PPT, Artikel, Ujian, dll)
- Marquee 50+ kampus se-Indonesia
- Pricelist 3 tab interaktif
- Auto-scroll testimoni slider
- FAQ accordion
- Tombol WhatsApp floating
- Tweaks panel (warna, kampus, hero, animasi)
- Fully responsive (4→2→1 kolom)
- Single file, plug-and-play

## Deploy

Upload `JokiBoy.html` + `tweaks-panel.jsx` ke satu folder, lalu deploy ke Vercel/Netlify.

Sebelum live:
- Ganti nomor WA di `wa.me/6282178880633` jika perlu
- Pasang Meta Pixel ID di `<head>`
